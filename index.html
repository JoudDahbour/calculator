How to Use

A calculator for basic arithmetic that solves chained operations left to right, one pair at a time.

Mouse

  • Tap digits, then an operator, then more digits
  • Press = for the result
  • C clears everything
  • Undo removes the last digit

Keyboard

  • 0–9 for digits
  • + - * / for operators
  • Enter for equals
  • Backspace to undo the last digit
  • Esc to clear
  • Use ← and → to scroll through a long number on the display

Good to know

  • Chaining example: 12 + 7 - 5 * 3 = gives 42
  • Dividing by zero shows a warning message!
